# AgentFace 生成式AI聊天界面

一个基于Vue 3和Element Plus的现代化AI聊天界面，灵感来源于ChatGPT的设计，提供了用户友好的AI交互体验。

## 功能特点

- 📝 类似ChatGPT的用户界面
- ⚡ 基于Element Plus的现代UI组件
- 🌈 Markdown支持，包括代码高亮
- 💬 聊天历史记录
- 🔧 响应式设计

## 安装依赖

```bash
npm install
```

## 开发运行

```bash
npm run dev
```

## 构建生产版本

```bash
npm run build
```

## 后端API连接

默认连接到 `http://localhost:9978` API端点，如需修改，请更新 `src/api/index.js` 文件中的 `baseURL`。

## 技术栈

- Vue 3
- Element Plus UI
- Axios
- Marked (Markdown解析)
- Highlight.js (代码高亮)
